# 游玩故事

玩家端 Web 应用默认地址是 `http://localhost:8000`。

## 游戏流程

1. 在浏览器中打开游戏。
2. 输入玩家名称。
3. 从故事列表中选择一个故事。
4. 创建新会话、通过会话码加入已有会话，或者在有可用存档时加载存档。
5. 在输入框中输入自由文本命令进行游玩。

## 多人模式

- 一名玩家可以创建会话并分享游戏码。
- 其他玩家可以在会话界面输入游戏码加入。
- 游戏会维护共享世界状态，同时保留每个玩家自己的状态数据。
- 如果浏览器里还保留会话令牌，刷新页面后通常可以重新连接到当前会话。

## 常用命令

- `look` 或 `l`：查看当前位置
- `inventory` 或 `i`：查看物品栏
- `help`、`h` 或 `?`：显示帮助
- `save [name]`：保存游戏
- `quit`、`exit` 或 `q`：离开当前游戏

## 界面说明

- 主消息区：显示叙事、对话、表单和选项
- 输入区：输入自由文本命令或动作
- 信息面板：显示状态、节点信息和物品栏
- 设置面板：显示游戏码、打字机效果开关、保存、导出历史、返回菜单和重新加载

## 重连行为

- 客户端会在浏览器存储中保存持久化玩家 ID。
- 刷新页面或短暂断线后，服务端可以尝试把你重新接回活动会话。
- 只要旧会话仍然存在，你通常会回到当前状态，而不是从头开始。

## 存档与读档

- 可以在命令输入框使用 `save`，也可以通过设置面板中的保存操作进行存档。
- 在故事和会话选择界面中可以看到可用存档。
- 读档主要通过会话选择界面完成，而不是在游戏内自由输入 `load` 命令。

## 导出历史

玩家端可以将当前可见的消息历史导出为 PNG 图片，这个过程完全在浏览器内完成。

## 语言

玩家界面支持英文和中文，并会在浏览器存储中记住当前语言设置。
