# 编辑器参考

## 菜单与状态

### File

- 新建故事
- 保存当前故事
- 打开故事列表
- 打开版本历史

### Edit

- 撤销
- 重做

### View

- simple view
- detailed view

### 图谱状态

图谱状态按钮会显示当前 `connections` 的状态：

- current
- stale
- missing

可以直接通过它触发重新编译。

## 面板

### 图谱画布

- 支持平移和缩放
- 显示故事节点与已编译的连接关系

### 节点检查器

- 编辑当前选中的节点
- 暴露节点字段和嵌套的故事内容

### 二级编辑器

- 用于聚焦编辑动作、物品或触发器

### 浮动工具面板

- 角色
- 参数
- lore
- 全局物品

### 版本历史

- 查看已保存版本
- 恢复旧版本

版本备份由后端保存在 `saves/story_versions/` 下。

## 保存语义

- 加载故事时，如果原始备份还不存在，会自动创建
- 保存通过后端 API 写入
- 后端会清洗故事 ID，并把 YAML 写到解析后的故事路径

## 快捷键与输入说明

菜单标签显示 `Ctrl+Z` 和 `Ctrl+Y` 用于撤销和重做。

实际使用时，请把菜单操作当作最可靠的工作流。如果你依赖键盘快捷键，请先在自己的环境中确认。

## 当前限制

- 编辑器不是玩家 Web 应用本身
- 故事试玩应在正常游戏界面完成，而不是依赖一个完整的内嵌预览模式
- 多文件故事在编辑器往返后可能被扁平化
- 旧 README 中有些能力描述比当前实现更宽泛
