SVG Character Card Prompt

This prompt is tuned for a minimalist branching-choice card game look with a geometric poster aesthetic: readable silhouette, flat color blocks, basic shapes, and a transparent SVG output that can drop straight into a square card UI.

Create a single standalone SVG illustration for a narrative choice card game character. Style goals: - minimalist medieval caricature with a geometric poster aesthetic - rely heavily on basic shapes: circles, rectangles, triangles, semicircles, diamonds, straight bands - strong silhouette readable at phone-screen size - flat vector shapes only, no raster effects - limited palette: 3 to 5 main colors - slightly whimsical, slightly ominous tone - portrait should feel like it belongs in a royal-court deck with roles like king, clown, doctor, bishop, executioner - facial features should be abstracted and graphic rather than naturalistic - prefer color-block separation over outlines - little to no stroke work Composition: - one character only - centered, mostly front-facing or 3/4 view - upper body only, like a bust or waist-up portrait - large head, simplified body, expressive hands - transparent background - enough negative space around the figure so it can sit inside a rounded square card frame SVG constraints: - output valid SVG code only - use clean grouped paths/shapes - avoid embedded images, filters, blur, gradients, masks, or text labels unless explicitly requested - keep the design crisp and scalable - prefer simple internal detail and 2D flat fills - avoid painterly detail, textured shading, anatomy realism, outlines, or too many bezier curves - use repeated shape rhythms where possible, like stacked circles, stripes, discs, and wedge forms - let shape overlap and negative space define the portrait instead of contour lines Character brief: - role: KING - mood: weary but commanding - age: late middle-aged - key props: crown, heavy robe, scepter - facial expression: suspicious, slightly tired eyes - visual motifs: gold crown points, deep burgundy robe, pale fur collar, ring on one hand - composition bias: oversized head, robe reduced to simple blocks, crown and scepter built from clean geometric pieces - minimalism bias: remove nonessential facial marks and keep each feature to the fewest shapes possible Output target: - viewBox suitable for a square card, for example 0 0 360 360 - return only the final SVG markup
Output: SVG only Background: transparent Tone: whimsical + uneasy
